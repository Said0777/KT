# ⌥ Cyber Store

Учебный проект — интернет-магазин техники Apple, разработанный в рамках итогового КТ курса «Проектный семинар».

**Стек:** Vue 3 (Composition API) · Pinia · Vue Router · Vite

---

## Запуск

```bash
npm install
npm run dev
```

Приложение откроется на `http://localhost:5173`.

Бэкенд (опционально): https://github.com/Arsen4ik/apple-store-backend/tree/build  
При недоступности бэкенда проект автоматически переключается на встроенные mock-данные.

---

## Структура

```
src/
├── assets/           # Глобальные стили
├── components/
│   ├── layout/       # AppHeader, AppFooter
│   └── ui/           # ProductCard, FiltersSidebar
├── views/            # Home, Catalog, Product, Cart, Favorites
├── stores/           # Pinia: товары, корзина, избранное
└── router/           # Vue Router (5 маршрутов)
```

---

## Реализованный функционал

- Каталог товаров с карточками и пагинацией
- Поиск в реальном времени (по названию и бренду)
- Фильтрация по цене, бренду и объёму памяти
- Сортировка по цене и рейтингу
- Корзина: добавление, изменение количества, удаление, оформление заказа
- Избранное: добавление и удаление товаров
- Страница товара с характеристиками и блоком отзывов
- Сохранение корзины и избранного через localStorage
- Адаптивная вёрстка (desktop + mobile)
